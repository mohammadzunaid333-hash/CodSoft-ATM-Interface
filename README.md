#CodSoft-ATM INTERFACE

## Overview

The **ATM Interface** is a console-based banking application developed in **Java** as part of my **Java Programming Internship at CodSoft**. The project simulates the core functionalities of an Automated Teller Machine (ATM) while demonstrating the practical application of Object-Oriented Programming (OOP) principles, clean code practices, and fundamental software development concepts.

The application provides a secure and interactive environment where users can perform essential banking operations through a structured, menu-driven interface. The primary objective of this project is to strengthen Java programming skills by implementing real-world banking functionalities in a modular and maintainable manner.


## Functional Overview

### User Authentication

Authenticates users using a predefined ATM PIN before granting access to banking operations.

### Balance Inquiry

Displays the current account balance in real time.

### Cash Deposit

Allows users to deposit funds into their account while updating the available balance.

### Cash Withdrawal

Processes withdrawal requests after validating sufficient account balance.

### Fund Transfer

Facilitates secure transfer of funds between accounts.

### Transaction History

Maintains and displays a record of user transactions for better financial tracking.
